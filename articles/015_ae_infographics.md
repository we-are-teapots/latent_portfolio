# Porcelain Gaiwan Tea Vessel Set

An elegant porcelain Gaiwan set hand-built using traditional Chinese techniques, featuring a lid, bowl, and saucer decorated with hand-painted cobalt blue underglaze designs. Fired at 1320°C to achieve translucent porcelain quality, this versatile tea brewing vessel showcases centuries-old craftsmanship adapted for modern tea enthusiasts.
