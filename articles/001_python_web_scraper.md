# Traditional Celadon Teapot Glazing

A masterwork celadon teapot featuring the ancient Korean tradition of jade-green glaze perfected over generations. This piece showcases the delicate iron-oxide reduction technique fired at 1280°C in a wood-burning kiln, creating the signature translucent green patina that has captivated tea enthusiasts for centuries.
